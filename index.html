Acerca de la Aplicación

AngularJS App sigue en desarrollo. La única operación disponible por el momento es la ecuación de primer grado y gráfica:

Ecuación de 1er Grado a base de Suma

x + =

Sólo para confirmar, tu ecuación es: {{s1}}x + {{s2}} = {{rs}}
Paso 1. Tomar la variable {{s1}}x.
Paso 2. Restar {{rs}} menos {{s2}}: {{rs-s2}}.
Paso 3. Unir los dos pasos anteriores: {{s1}}x = {{rs}} - {{s2}}.
Paso 4. Substituir la operación: {{s1}}x = .
Paso 5. Dividir el resultado sobre la x: {{((rs-s2)/s1).toFixed(2)}}.

En la ecuación {{s1}}x + {{s2}} = {{rs}}, con base en una suma; x tiene un valor de {{((rs-s2)/s1).toFixed(2)}}.

Ecuación de 1er Grado a base de Resta

x - =

Sólo para confirmar, tu ecuación es: {{r1}}x - {{r2}} = {{rr}}
Paso 1. Tomar la variable {{r1}}x.
Paso 2. Sumar {{rr}} más {{r2}}: {{(rr*1)+(r2*1)}}.
Paso 3. Unir los dos pasos anteriores: {{r1}}x = {{rr}} + {{r2}}.
Paso 4. Substituir la operación: {{r1}}x = .
Paso 5. Dividir el resultado sobre la x: {{(((rr*1)+(r2*1))/r1).toFixed(2)}}.

En la ecuación {{r1}}x - {{r2}} = {{rr}}, con base en una resta; x tiene un valor de {{(((rr*1)+(r2*1))/r1).toFixed(2)}}.
