Sobre AngularJS

AngularJS es un framework MVC de JavaScript para el Desarrollo Web que permite crear aplicaciones SPA (Single-Page Applications).

Acerca de la Aplicación

AngularJS App sigue en desarrollo. La única operación disponible por el momento es la ecuación de primer grado:

x + =

Paso 1. Tomar la variable {{n1}}x.
Paso 2. Restar {{r}} menos {{n2}}
Paso 3. Unir los dos pasos anteriores así: {{n1}}x = {{r}} - {{n2}}
Paso 4. Substituir la operación: {{n1}}x =
Paso 5. Dividir el resultado sobre la x: {{(r-n2)/n1}}

En la ecuación {{n1}}x + {{n2}} = {{r}}; X tiene un valor de {{(r-n2)/n1}}.
